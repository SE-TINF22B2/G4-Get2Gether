Event erstellen

Erstelle ein neues Event für deine nächste Party, Urlaub oder sonstige Veranstaltung.

Event erleben

Alles fertig geplant? Genieße dein Event und feiere mit deinen Freunden!

Gemeinsam Events planen

Plane zusammen mit deinen Freunden events.

Den Überblick behalten

Siehe welche Events als Nächstes anstehen und behalte den Überblick wo, wann und mit wem deine Events stattfinden.

Völlig kostenlos

Get2Gether ist kostenlos nutzbar! Spare dir das Geld für deine Events.

Mehr Funktionalität mit Widgets

Erweitere deine Eventplanung mit praktischen Widgets.

Karte

Markiere Orte auf der Karte, sodass jeder den Weg zum Event findet.

Einkaufsliste

Habt ihr alles benötigte gekauft? Seht immer auf einen Blick, was bereits gekauft wurde und was noch fehlt.

Ausgabenverteilung

Seht immer auf einen Blick, wer was bezahlt hat und wer noch etwas schuldig ist.

Fahrgemeinschaft

Organisiert Fahrgemeinschaften mit Fahrer und Mitfahrer. So muss keiner zum Event laufen!
